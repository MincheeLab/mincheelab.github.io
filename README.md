www
===

Minchee Foundation website

original Munter theme from http://www.blacktie.co/2013/10/munter-one-page-theme/


To install and run dev server:

```bash
npm install
cd src
bower install
gulp serve
```

To push in prod:

```bash
cd src
gulp build
cd ..
git commit -a -m "your commit message here..."
git push
```
